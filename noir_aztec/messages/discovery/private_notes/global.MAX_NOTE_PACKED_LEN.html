Global MAX_NOTE_PACKED_LEN

pub global MAX_NOTE_PACKED_LEN: u32;

The maximum length of the packed representation of a note's contents. This is limited by private log size, encryption overhead and extra fields in the message (e.g. message type id, storage slot, etc.).
