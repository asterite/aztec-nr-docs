aztec-nr - noir_aztec::messages::logs::note

Module note

Functions

  • compute_note_log
  • compute_partial_note_log
  • encode_and_encrypt_note
    Sends an encrypted message to recipient with the content of the note, which they will discover when processing private logs.
  • encode_and_encrypt_note_unconstrained
    Same as encode_and_encrypt_note, except encryption is unconstrained. This means that the sender is free to make the log contents be whatever they wish, potentially resulting in scenarios in which the recipient is unable to decrypt and process the payload, leading to the note being lost.
  • encode_and_encrypt_note_and_emit_as_offchain_message
    Same as encode_and_encrypt_note_unconstrained, except the note is emitted as an offchain message instead of a private log.
