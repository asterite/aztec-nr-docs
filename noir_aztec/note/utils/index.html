aztec-nr - noir_aztec::note::utils

Module utils

Functions

  • compute_note_hash_for_read_request
    Returns the note hash that must be used to issue a private kernel read request for a note.
  • compute_note_hash_for_nullify
    Returns the note hash that must be used to compute a note's nullifier when calling NoteHash::compute_nullifier or NoteHash::compute_nullifier_unconstrained.
  • compute_note_hash_for_nullify_from_read_request
    Same as compute_note_hash_for_nullify, except it takes the note hash used in a read request (i.e. what compute_note_hash_for_read_request would return). This is useful in scenarios where that hash has already been computed to reduce constraints by reusing this value.
  • compute_siloed_note_nullifier
    Computes a note's siloed nullifier, i.e. the one that will be inserted into the nullifier tree.
